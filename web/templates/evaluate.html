End-to-end uncertainty evaluation for PCAM (patch-level). This page covers predictive performance, calibration, uncertainty quality, selective prediction, and shift/OOD robustness.

0. How to use this page

Recommended order: (1) choose model/method, (2) run Full thesis evaluation, (3) run Shift/OOD evaluation, (4) run All evaluations for one combined thesis summary.

  • Why this is needed: uncertainty should be evaluated beyond accuracy, including calibration and behavior under distribution shift.
  • Dataset scope: current implementation is PCAM-only; slide-level output is provided as a PCAM proxy aggregation.
1. Model

Select a trained run and uncertainty method.

  • Confidence (single pass / MSP): crude baseline only; typically overconfident and not suitable as primary uncertainty evidence.
  • MC Dropout: stochastic test-time passes for epistemic uncertainty approximation.
  • Deep Ensemble: stronger uncertainty baseline using multiple trained members (if available).

8. Methods and sources

References used by this evaluation tab and why each is included.

  • Temperature Scaling (Guo et al., 2017) — standard post-hoc calibration baseline.
  • MC Dropout (Gal & Ghahramani, 2016) — stochastic uncertainty approximation.
  • Deep Ensembles (Lakshminarayanan et al., 2017) — strong uncertainty baseline.
  • AURC (Torch-Uncertainty) — selective prediction safety metric.
  • AUROC (scikit-learn) — discrimination and OOD/error detection ranking metric.
  • Brier Score — proper scoring rule for probabilistic predictions.
  • Torch-Uncertainty — metric reference framework used for uncertainty evaluation context.
  • Dataset shift in digital pathology (Thagaard et al., 2020) — motivation for robustness under real shifts.
  • OOD in digital pathology (Linmans et al., 2022) — motivation for pathology-specific OOD analysis.

Additional related papers (from current literature review)

  • Bhattacharyya et al. (2025) — aleatoric/epistemic uncertainty for tumor cellularity regression.
  • Dolezal et al. (2022) — dropout-based uncertainty thresholding to defer low-confidence predictions.
  • Park et al. (2025, UDLS) — uncertainty-guided label smoothing for MIL calibration.
  • Park et al. (2025, MSI-SEER) — deep Gaussian process uncertainty in weakly supervised WSI setting.
  • Jordahn et al. (2026) — practical heteroskedastic regression baseline.
  • Kinakh et al. (2025) — Bayesian multimodal posterior sampling for histopathology segmentation.

These papers are cited as related work. Some methods are not yet implemented in this repository's current PCAM pipeline.

9. Metric glossary (quick interpretation)
  • ROC-AUC / PR-AUC: discrimination/ranking metrics; useful but insufficient alone for per-prediction confidence validity.
  • ECE: probability calibration error. Lower is better.
  • NLL / Brier: proper scoring rules for probabilistic predictions. Lower is better.
  • AURC: selective prediction risk-coverage area. Lower is better.
  • OOD AUROC / AUPRC: ability to separate in-distribution and shifted samples via uncertainty. Higher is better.
